# JavaScript Problem Practice

This folder contains my JavaScript practice problems, with each problem solved in its own file.

I created these files to practice problem solving, improve my JavaScript fundamentals, and revise common coding interview style questions.

## Folder Structure

Each file is standalone and usually includes:

- the problem statement or a short note
- a JavaScript solution
- a `console.log(...)` example at the bottom

## How To Run

If you want to run any file, make sure you have [Node.js](https://nodejs.org/) installed and use:

```bash
node Problem1.js
node Problem10.js
node Problem25.js
```
